Informations du participant

Cette version est prête pour une connexion à Google Sheets via Google Apps Script. Le site ne montre plus le mot de passe formateur sur la page d’accueil.
Version 3 : si vous renseignez un endpoint web dans la configuration du code, les résultats pourront être envoyés automatiquement vers un stockage centralisé compatible avec GitHub Pages.
